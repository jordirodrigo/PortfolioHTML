Portfolio 3D Interactivo – Web

Este proyecto es un portfolio web que permite explorar mis trabajos de animación, rigging y modelado 3D de manera interactiva. Está desarrollado usando HTML, CSS y JavaScript, y utiliza la librería Three.js para mostrar y manipular los modelos 3D directamente en el navegador.

Incluye una interfaz limpia y responsive, con secciones para mostrar mis proyectos destacados y acceso a mi información de contacto. Los modelos se pueden rotar, acercar y desplazar para verlos desde diferentes ángulos, ofreciendo una experiencia dinámica más allá de un portfolio estático.

Para ejecutar el proyecto basta con abrir index.html en un navegador moderno. Algunos navegadores pueden requerir ejecutarlo desde un servidor local para que los modelos 3D se carguen correctamente.

El portfolio está pensado como una herramienta profesional para mostrar mis habilidades en 3D y desarrollo web, y está inspirado en la estética de videojuegos modernos como Zelda BOTW o pokemon , buscando un estilo visual elegante y envolvente.

El proyecto se distribuye únicamente como portfolio personal y no debe ser redistribuido sin permiso.
